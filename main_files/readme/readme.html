Adding your logo

Theme comes with dummy logo ( Laura & Smith ) which can be replaced easily.

Open logo.png and enter your company or business name and save it as a logo.png in images folder.

If you already exist the logo, then make sure your logo sizes to be 225px wide and 55px height, this is not compulsory but try to use these sizes to ensure the look and feel of your layout

Place your logo in images folder with the name called logo.png , now your logo will be displayed automatically on the top left of the page.

Adding an Email ID in Contact page

Template comes with ready to use contact form, what you have to do is, just adding your email id to recieve emails from your website.

open sendEmail.php: and find youremailid@domain.com and replace it with your email id.

CSS Files

I'm using 4 CSS files mainly in this theme, the others are supporting files.

  1. layout.css: contains all DIVs which construct the layout prototype, using div class="one_half", div class="one_half_last" div class="one_third" and div class="one_third_last" etc.
  2. jqueryslidemenu.css: is used for top dropdown menu, if you would like to make changes to the drop down menu links, open jqueryslidemenu.css and find bellow code. 
     
                    	.jqueryslidemenu {
                            position:relative;
                            z-index:99;
                        }
                        .jqueryslidemenu ul {
                            margin: 0;
                            padding: 0;
                            list-style-type: none;
                        }
  3. jquery.bxslider.css are used for Gallery slider.
  4. style.css: contains all of the specific stylings for the page skin or look and feel. The file is separated into several sections like the bellow as sample: 

     
                        body {
                            color: #EBEBEB;
                            font-family: Arial, Helvetica, sans-serif;
                            font-size: 13px;
                            line-height: 18px;
                            margin:0px;
                            padding:0px;
                        }
                        p, ul, ol, dl, table, fieldset, blockquote, pre, code {
                        }

Jquery Files

This theme imports 5 Javascript files mainly, others are supporting files.

  1. jquery.min: is a Javascript library that greatly reduces the amount of code that you must write.
  2. jqueryslidemenu: is used for menu
  3. jquery.localscroll-1.2.7-min & jquery.scrollTo-1.4.2-min: is used for Page Scroll
  4. jquery.bxslider: is used for Gallery Page
  5. contact.js: used for Venue page.
